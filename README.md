# Расчет параметров прямоугольника и окружности
Геометрические фигуры. Реализовать функции вычисления параметров для двух следующих фигур:
    
Прямоугольник — периметр, площадь, длина диагонали;
    
Окружность — длина окружности, площадь круга, площадь кругового сектора.
