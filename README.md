# Расчет параметров прямоугольника и круга

Реализовать функци вычисления для следующих фигур:

Прямоугольник - периметр, площадь , длина диагонали;

Круг - длина окружности , площадь круга , площадь кругового сектора

## Описание 
Проект выполнен в рамках Лабораторной работы №1.

